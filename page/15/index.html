資安名詞-Demilitarized Zone (DMZ)

Author published on 2020-01-26
DMZ為外部網路使用者提供有限度的連線存取，讓外部使用者能最小限度使用組織所提供的公開資源(如網站)，並且與內部網路(LAN)區隔。一般來說，會使用Firewall隔出DMZ區。同時，DMZ區主機亦會限制存取內部網路資源，避免DMZ區受駭時，造成內網資安的威脅。 在現在的資安觀念上，除了基本的網段區隔外，零信任也是重要的觀念。 延伸閱讀 資訊安全-內網橫向移動(Lateral Movement)介紹 工具介紹-Weevely (kali linux web shell產生工具)

名詞介紹-Domain Name Service (DNS)

Author published on 2020-01-25
DNS提供將Domain name進行解析成IP的服務，需使用53/TCP、53/UDP(通常為UDP)進行連線。 DNS查詢 在電腦在需要連線Domain時，會優先查找主機內的hosts file，並利用該資料進行連線。如果沒有找到，將再對所設定的DNS提出Domain查詢的請求。 Linux 先找/etc/hosts windows 先找 C:\windows\System32\drivers\etc\hosts 依RFC 3330 localhost 會指到127.0.0.1 DNS設定(Linux) http://www.geekpills.com/operating-system/linux/configure-master-slave-bind-dns-in-linux 延伸閱讀 DNS amplification attack (DNS放大攻擊)簡介 10 DNS Attacks Types and The Mitigate Steps – 2020 DNS tunneling 參考資料 https://tools.ietf.org/html/rfc3330 https://www.cloudflare.com/learning/dns/what-is-dns/ https://support.microsoft.com/zh-tw/topic/microsoft-tcp-ip-%E4%B8%BB%E6%A9%9F%E5%90%8D%E7%A8%B1%E8%A7%A3%E6%9E%90%E9%A0%86%E5%BA%8F-dae00cc9-7e9c-c0cc-8360-477b99cb978a
