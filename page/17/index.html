Windows - 利用Kali Linux找回登入密碼

Author published on 2020-08-15
最近一次機會下，需要幫朋友登入忘記密碼的筆電。原本想利用常見的手法，用windows 重灌光碟把cmd.exe改成放大鏡，但居然因為找不到相應的版本而沒有辦法使用。 換個思路與其找相同版本的光碟，重點在將cmd.exe換成放大鏡這個步驟，所以決定改利用Linux Live USB來解決這個問題。 在將cmd.exe改成放大鏡時，我遇到另一個坑，就是朋友筆電中的cmd.exe不見了！！於是改採用Powershell來替代，進入windows powershell資料夾，把powershell複制出來，把powershell.exe改成放大鏡，照樣可以進行。 以下為執行步驟 首先使用Kali Linux 所提供的方法安裝完Live USB。 利用USB 開機，登入Kali linux。 把原本的c槽，Mount到指定的資料夾下。 將放大鏡程式改名成.bak。 再將powershell.exe改成放大鏡。 將電腦正常開機，在登入頁面點開放大鏡。 在powershell視窗中利用net指令將忘記密碼的帳戶更新密碼 完成上述步驟後就可以照之前的流程使用net指令完成改密碼。

工具介紹-Tcpdump指令介紹

Author published on 2020-08-09
tcpdump 指令可以用來監聽網卡的封包，是相當實用的 Linux 指令。 確認是否有安裝 which tcpdump 安裝tcpdump # yum install -y tcpdump 參數介紹 -d 看可用的介面卡 -i 選擇特定的介面卡 tcpdump -nnSX port 443 取得 HTTPS port 443 的流量。 延伸閱讀 工具介紹-Microsoft Network Monitor 參考資料 https://opensource.com/article/18/10/introduction-tcpdump https://www.tcpdump.org/manpages/tcpdump.1.html
