Kali Linux 升級跟版本確認

作者
 2022-05-25  約 75 字   預計閱讀 1 分鐘 
目錄
  • 升級指令
  • 確認版本指令

Kali Linux 會定期更新，有時會出一些不錯的功能，可以透過以下指令來進行更新。

升級指令

sudo apt update
sudo apt full-upgrade -y

確認版本指令

確認版本可以使用以下兩個指令

方法 1

lsb_release -a

方法 2

cat /etc/os-release
