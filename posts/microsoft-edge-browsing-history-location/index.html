說明

Edge的瀏覽記錄位置如下，檔案名稱為History，以SQLite的型式儲存。

C:\Users\\AppData\Local\Microsoft\Edge\User Data\Default\

取得資料後可以使用讀取SQLite的工具，如SQLite Reader去讀取資料。

如果有在使用 VS Code 的話，可參考 vscode-sqlite 這個 extension 。
